# a log parser in clojure

A Clojure app to parse a log file. Why? Just to play arounnd wit clojure :)

## Requirements

1) Java JDK version 6 or later
2) [Leiningen](https://leiningen.org/)

## Usage

```
lein run LOG_FILE
```

## Testing

```
lein test
```
